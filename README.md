 CodeAlpha_Image-gallery
Responsive Image Gallery

A modern and responsive image gallery built using HTML, CSS, and JavaScript. The project provides a clean user interface with interactive image navigation, category filtering, hover animations, and a fullscreen lightbox for an enhanced viewing experience. It is fully responsive and works seamlessly across desktop, tablet, and mobile devices.

✨ Features
Responsive gallery layout for all screen sizes
Interactive lightbox for fullscreen image preview
Previous and Next image navigation
Category-based image filtering
Smooth hover animations and transitions
Keyboard navigation support (Arrow Keys & Escape)
Clean, modern, and user-friendly interface
Optimized with pure HTML, CSS, and Vanilla JavaScript (No frameworks)
🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6)

🚀 How to Run
Download or clone the repository.
Open the project folder.
Launch index.html in any modern web browser.
Browse images, apply filters, and use the lightbox navigation.
